# Twitter High-Res Video Downloader (twi-hai-dl4K)

Twitter (X) の動画を可能な限り最高画質でダウンロードするための Chrome 拡張機能です。

## 特徴

- **最高画質でダウンロード**: 利用可能な最高解像度の動画 URL を自動的に取得します。
- **シンプルな操作**: ツイートのアクションバーに追加されるダウンロードボタンをクリックするだけ。
- **スマートなファイル名**: 投稿者名、ツイート本文、ツイート ID を組み合わせた、整理しやすいファイル名で保存します。
- **高画質・GIF対応**: 動画だけでなく、GIFアニメーションのダウンロードにも対応しています。

## インストール方法

この拡張機能は Chrome ストアには公開されていないため、以下の手順で手動インストールしてください（デベロッパーモード）。

1. このリポジトリをクローンまたはダウンロードし、適当なフォルダに解凍します。
2. Chrome ブラウザを開き、アドレスバーに `chrome://extensions/` と入力してアクセスします。
3. 右上の「デベロッパーモード」スイッチをオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします。
5. 解凍したフォルダの中にある `src` フォルダを選択します。

## 使い方

1. Twitter (X) を開き、動画が含まれるツイートを表示します。
2. ツイートの下部（いいねやリツイートボタンの並び）に、ダウンロードアイコン（下矢印）が表示されます。
3. アイコンをクリックすると、動画のダウンロードが開始されます。

## ファイル名の形式

ダウンロードされるファイルは以下の形式で保存されます。

```
[表示名]_[本文の一部]_[ツイートID].mp4
```

- 特殊文字やファイル名に使えない文字は自動的に除去・置換されます。
- 長すぎる名前は適切に切り詰められます。

## 技術的な詳細

- **Manifest V3**: 最新の Chrome 拡張機能の仕様に準拠しています。
- **Content Script & Injected Script**: ページ内のコンテキストにアクセスするために、スクリプトインジェクションを使用しています。
